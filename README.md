Customer Analysis Project
This project focuses on analyzing customer behavior using Python and Excel to extract meaningful business insights.
Project Overview
In this project:
* Raw data was cleaned and prepared using Python (Pandas)
* Customer-level analysis was performed
* Key metrics such as total revenue and order frequency were calculated
* Results were visualized using an Excel dashboard
Key Insights
* A small number of customers generate a large portion of total revenue
* Customers with the highest revenue are not always the most frequent buyers
* Customer value is more important than purchase frequency for business growth
Tools Used
* Python (Pandas)
* Excel (Pivot Tables & Dashboard)
Project Files
* customer_analysis.ipynb → Data analysis using Python
* customer_analysis.xlsx → Final dashboard
* dashboard.png → Dashboard preview
